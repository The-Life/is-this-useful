# Evidence Sources

Prefer:

1. Primary docs and source code
2. GitHub issues, PRs, releases, commit activity
3. Benchmarks with methodology
4. Pricing/access docs
5. Real after-use reports
6. Community critiques with specifics

Be skeptical of launch threads, star spikes, repost chains, generic "game changer" claims, affiliate reviews, and benchmark screenshots without methodology.

For Reddit/community reaction, distinguish:

- after-use reports: high value
- installation struggle reports: medium value
- launch/promo reposts: low value
- jokes/memes: low value unless sentiment is the question
