# Contributing

Thanks for helping improve Is This Useful?

Good contributions:

- clearer decision rubrics
- examples from different domains
- install notes for more agents
- evidence grading improvements
- tests that catch broken skill packaging

## Local validation

```bash
python3 scripts/validate.py
```

## Principles

- Decision first.
- Protect attention.
- Prefer small pilots over big adoption.
- Do not turn hype into backlog by default.
